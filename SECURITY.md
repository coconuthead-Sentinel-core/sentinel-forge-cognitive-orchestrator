# Security Policy

## Scope
This repository contains the Sentinel Forge Cognitive AI Orchestration Platform source, paperwork, and local-development runtime assets.

## Reporting a Vulnerability
- Do not open a public issue for a sensitive security finding.
- For sensitive reports, contact the repository owner through the GitHub account `coconuthead-Sentinel-core` and request a private disclosure channel before sending exploit details.
- For non-sensitive hardening suggestions, you may open a normal GitHub issue and label it as security-related.

## What to Include
- affected file paths, endpoint names, or workflows
- reproduction steps
- impact description
- whether the issue is exploitable in mock mode, live Azure mode, or both
- any suggested remediation

## Response Expectations
- acknowledgment target: within 7 calendar days
- initial triage target: within 14 calendar days
- remediation timing depends on severity, exploitability, and release risk
