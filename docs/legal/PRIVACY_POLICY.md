# Privacy Notice

## Scope
This document describes the current privacy posture for the Sentinel Forge repository review release.

## Current Operating State
- The repository is being distributed for technical review and local development.
- No public hosted SaaS service is active from this release.
- No public account signup or payment flow is active from this release.

## Data Handling For The Current Release
- Source code, documentation, and test assets are stored in the repository.
- If a reviewer runs the software locally, data handling occurs in that reviewer's local environment.
- Optional Azure integrations exist in code, but they are not required for the local mock-mode review flow.

## No Public Service Data Collection Claim
Because no public hosted service is active from this release, this repository does not claim a live production privacy program for public end users.

## Contact
Questions about this repository release may be directed to the repository owner through the GitHub namespace `coconuthead-Sentinel-core`.

## Future Activation Rule
If Sentinel Forge is later launched as a hosted public service, this notice must be replaced with a deployment-specific privacy policy that reflects real operators, environments, storage locations, retention rules, and customer support channels.
