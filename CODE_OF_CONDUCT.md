# Code of Conduct

## Purpose
Sentinel Forge is maintained as a respectful technical workspace for contributors, reviewers, and users.

## Expected Behavior
- Be respectful and direct.
- Critique code and ideas without attacking people.
- Keep discussions useful, evidence-based, and relevant to the work.
- Support accessibility and neurodivergent-friendly communication.

## Unacceptable Behavior
- Harassment, discrimination, or intimidation
- Personal attacks
- Deliberate disruption of technical collaboration
- Publishing private or sensitive information without consent

## Enforcement
The repository owner may remove content, restrict participation, or decline contributions that violate this policy.

## Reporting
For conduct concerns, contact the repository owner through the `coconuthead-Sentinel-core` GitHub account.
